Icono

El icono es una imagen pequeña que representa una aplicación, sitio web o archivo. Se utiliza para identificar visualmente el contenido y facilitar su reconocimiento. El icono puede ser un logotipo, un símbolo o una representación gráfica relacionada con la función o el propósito del elemento que representa.

Meta Task

La meta task es una tarea que se establece como objetivo principal para lograr un resultado específico. Es una tarea que se enfoca en alcanzar un objetivo a largo plazo y puede involucrar varias sub-tareas o actividades para su cumplimiento. La meta task ayuda a organizar y dirigir los esfuerzos hacia la consecución de un resultado deseado.
